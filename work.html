Meet App

A React-based web application that allows users to view and filter events by city, track the number of events, and explore event genres. This app integrates with the Google Calendar API and offers a mobile-friendly design.

See the App in action

Project Details

This web app allows users to:

  • View and filter events by city.
  • Adjust the number of events displayed.
  • Explore event genres via charts.
  • Responsive design for desktop and mobile.

Tech Used: React, Google Calendar API, JavaScript, CSS, HTML, Bootstrap
